# 穿越者论坛主题说明

本版本已经把 Modern BBS 改成“临高元老院”穿越者论坛主题：

- 前台：议事厅、发起讨论、元老回复、档案检索
- 分区：元老院公告、工业建设、农业与物资、医疗卫生、军务安保、航海贸易
- 后台：元老院后台、身份管理、议案管理、回复管理、分区管理
- 风格：暖色纸张 + 深色海图卡片，偏档案/议事厅感觉

继续修改入口：

- 论坛名：src/main/resources/templates/fragments.html
- 首页文案：src/main/resources/templates/index.html
- 分类分区：src/main/resources/data.sql
- 样式配色：src/main/resources/static/css/app.css
- 数据库密码：src/main/resources/application.yml
